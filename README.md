
# I'm Not A Number
is a digital memorial project designed to humanize the victims of the ongoing conflict in Gaza. Instead of viewing lives as mere statistics, this interactive visualization allows users to explore individual stories through a digital "starfield" of memories.

> Every dot represents a human life — not a statistic.

# 🌍 About the Project
I'm Not a Number is an interactive web experience that visualizes real human data as a constellation of points
> Each point on the screen represents a person defines by his
-  A name
 -  An age
 -  A story
-  A life that mattered
The goal of this project is to transform cold data into human emotion and awareness.


# 🕊️ Experience the Memorial
Behind every number is a life, a dream, and a story. This interactive visualization is designed to bridge the gap between raw data and human empathy. 
**[Click here to experience the memorial live](https://remembre-us.netlify.app/)**



# ✨ Features
- 🎨 Real-time canvas visualization of thousands of points
- 🔍 Search by name
- 🎚️ Filters (age range + gender)
- 🖱️ Hover interaction with live info cards
- 🎯 Click to focus on a person
- 🌌 Smooth camera zoom/focus animation
- 🎧 Optional background ambient sound
- 💬 Rotating emotional captions
- 📱 Responsive fullscreen experience

# 🛠️ Tech Stack
- React (Hooks: useState, useEffect, useRef, useMemo)
- HTML5 Canvas API
- JavaScript (ES6+)
- CSS (inline + minimal styling)
- Audio API (for background sound)

# 🚀 How to Run Locally
## 1. Clone the repo
git clone [https://github.com/Hibabg21/im-not-a-number]

## 2. Install dependencies
npm install

## 3. Run development server
npm run dev

# 📊 Data Source
This project uses publicly available records. 
*Dataset: `/public/killed-in-gaza-v3.json`*

# 🎮 How to Use
- Move mouse to explore the field of points
- Hover over a point to see details
- Click on a point to focus on a person
- Use search bar to find a name
- Filter by age or gender
- Toggle sound on/off
- Press “Reset View” to return

# 🎯 UI States
## Intro Screen : 
    start experience with or without sound
## Main Canvas :
    interactive visualization
## Hover Card :
    person details
## Caption System :
    rotating emotional messages
